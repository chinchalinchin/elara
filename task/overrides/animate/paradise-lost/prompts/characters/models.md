# Satan

Create a character model for John Milton's Satan in Paradise Lost. It should be a single model, front and back. It is meant to be a reference for a character design, not a standalone piece. Satan should be handsome and alluring, with angular and striking features. He should have a dignified bearing. Leathery wings, wearing roman-esque armor and flowing togas. The rendering should be in the style of a 17th century ballad illustration.
