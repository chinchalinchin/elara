# elara

[elara.chinchalinchin.com](https://elara.chinchalinchin.com)

## Quickstart

```bash
python3 -m venv venv
source ./venv/bin/activate
pip install -r requirements.txt
cd src 
make html
```

### Build Dependencies

To render the mermaid diagrams in `svg` format, the mermaid CLI must be available.

```bash
npm install -g @mermaid-js/mermaid-cli
```
