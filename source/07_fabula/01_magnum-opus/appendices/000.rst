.. VAN RENSSELAER FAMILY TREE DEVELOPMENT
..
.. GOAL: Using mermaid, develop and complete the exhaustive family tree of the van Rensselaer dynasty, given below, from the first patroon up to the present day.
..
.. TERMS
..
..      - Dutch Branch (Cralo Branch): Family branch that remained in Holland.
..      - American Branch: Family branch that relocated to America.
..          - Elder Branch: Lords of the Manor
..          - Younger Branch: Lords of Claverack
..
.. TIMELINE
.. 
..      The timeline should only include major historical events (relative to the family). 
..
.. STYLE
.. 
..      - Use the principles of Material UI when making updates to the styles.
..
.. COLOR
..
..      ** GUIDELINES**
..
..      - Use the principles of Color Theory and Color Wheels when selecting new colors.
..
..      ** GRADIENT**
..
..      - Color gradient is used to visualize the descent of the title from the earlier generations (Darker) to the later generations (Lighter).
..      - Color gradient applies to Primary, Secondary, Tertiary and Quaternary schemes. Quinary only has a single shade.
..
..      **PALETTE**
..
..      - Primary (Purple): Represents the (American) Patroon Lineage (Lords of the Manor). 
..          - Starting Shade (Hex Code): #311B92
..      - Secondary (Amber): Represents Family Members (Wives, siblings, younger branches).
..          - Starting Shade (Hex Code): #E65100
..      - Tertiary (Blue): Represents the (Dutch) Patroon branch.
..          - Starting Shade (Hex Code): #283593
..      - Quaternary (Orange): Represents the Younger branch (Lords of Claverack)
..          - Starting Shade (Hex Code): #E65100
..      - Quinary (Teal): Represents Special Individuals. 
..          - Shade (Hex Code): #00897B
..      - External (White): Connections of female descendants of male lines to their spouses.
..
.. NODES
..
..      **Shapes**
..
..      - Circle (): Only Killiaen van Rensselaer, the 1st Patron
..      - American Patroons {{ }}: Hexagons.
..      - Dutch Patroons ([]): Stadium-shaped Node
..      - Generic Male of Male Descent []: Rectangles
..      - Generic Female of Male Descent {}: Diamonds
..      - Generic Male of Female Descent [/\]: Inverse Trapezoid
..      - Generic Female of Female Descent [\/]: Trapezoid
..      - External Node: Plain Rectangle
..
..      Patroon shapes take precedence. American Patroonship takes precedence over Dutch Patroonship.
..
..      **Labels**
..  
..      - Name
..      - Birth & Death Years
..      - Patroonship, if applicable.
..      - Directorship, if applicable.
..      - Regency, if applicable.
..  
..      No other labels are to be introduced. DO NOT ADD NEW LABELS.
..
..      - Exception: Jesuit Priests
..      - Exception: Known Aliases
.. 
.. RELATIONS
.. 
..      **Marriage**
..
..      - The relations between husbands and wives should include the year they were married (if they were married), prefixed with "m. <year>". 
..      - Dotted lines indicate the marriage of cousins.
..      - Thick lines with a dot indicates the marriage of a daughter to an external family. Only the last name of the external connection should be listed.
..      
.. GUIDELINES
..
..      - If a date is unknown, approximate it and use a "?" to denote uncertainty.
..      - Generations should always be aligned to descending rows.
..      - Do not remove nodes from the tree.
..      - DO NOT REMOVE NODES FROM THE TREE!
..      - **DO**. **NOT**. **REMOVE**. **NODES**. **FROM. **THE*. **TREE*.
..      - Do not alter labels unless explictly instructed to.
..
.. TASK
..
..      - Select a starting shade for the Hendrick van Rensselaer - Catharina van Burgh Branch (the starting point of the Younger Branch) that is close to the Amber shade of that node. Then apply the gradient down through the Younger Branch. 
..
.. OUTPUT
.. 
..      Your output does not need to be RST. Output the mermaid diagram in whatever native format is easiest, and include a list of changes that were applied.
